Unix Shell Command Cheat Sheet

Learn Basic Unix Shell Commands

I'm just starting to learn how to use the UNIX Shell. At first it was a little bit intimidating but the more I use it the more I realize how useful it is. Here I have compiled a list of some basic, useful commands for the Unix Shell.

Command Description Example(s)
cd [options] [directory] Change directory. You can use: 
$ cd ..
to go up one directory.		 
$ cd Documents
ls [options] [filename ...] List files 
$ ls
Documents/
notes.txt
index.html
